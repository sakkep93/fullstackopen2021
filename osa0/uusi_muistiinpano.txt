server->*database: {"content": "hello world", "date": "2021-05-19T15:23:51.700Z"}
database-->server: successfully added new note
server-->browser: HTTP 302 Redirect\nhttps://studies.cs.helsinki.fi/exampleapp/notes

note over browser:
after a successful addition of a new note,\nthe user is redirected to /examples/notes,\nwhich initiates a new sequence of events
end note

browser->server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/notes
server-->browser: HTML-code

server-->browser: main.css
browser->server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.js
server-->browser: main.js

note over browser:
the browser starts to execute js code,
which requests JSON-data from the server
end note

database-->server: notes data
